Omslagssökning

Open Library Covers Ingen nyckel krävs

Hämtar bokomslag direkt via ISBN från Internet Archives öppna bibliotek. Fungerar bäst för böcker som redan har ISBN ifyllt. Helt gratis, ingen konfiguration.

Google Books — högupplöst omslag Ingen nyckel krävs

Uppgraderar automatiskt thumbnails från Google Books till fullstorlek. Använder samma data som "Hämta metadata" men begär högre upplösning. Noll extra API-anrop.

Hardcover {% if hardcover_token_set %} ✓ Ansluten {% else %} Valfri token {% endif %}

Modern bokdatabas med bra täckning för nyare böcker och serier. Fungerar utan nyckel men en API-token ger bättre rate limits.

{% if hardcover_token_set %} {% endif %}
Env: COLOPHON_HARDCOVER_API_TOKEN
Hur skaffar jag en token?
  1. Gå till hardcover.app/account/api
  2. Skapa konto (gratis)
  3. Kopiera din API-token

Gratis beta — valfritt, fungerar utan token.

Beta-API — kan ändras utan förvarning. Token gäller 1 år.

Wikidata / Wikimedia Commons Ingen nyckel krävs

Söker bokomslag via Wikidata och Wikimedia Commons. Bäst för klassiker, kända verk och public domain-böcker. Kräver ISBN.

DuckDuckGo bildsökning Ingen nyckel krävs

Sista fallback — söker bokomslag via DuckDuckGo. Inofficiellt API som kan sluta fungera. Bra som sista utväg för böcker som inte hittas i bokdatabaser. Respekterar rate limiting.
